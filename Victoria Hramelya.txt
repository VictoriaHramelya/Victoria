1. ls - просмотр папок в текущей директорим
2. cd <Имя папки> + tab в процессе ввода имени, если нужно, чтобы оно подставилось автоматически
3. git status - получения статуса файлов в репозитории
4. git add . - перенос файл в директорию подготовленных файлов
5. git commit -m "комментарий" - фиксация изменений
6. git push - публикация изменений на сервере
7. git log - просмотр истории изменений (в обратном порядке)
